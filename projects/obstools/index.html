    FM-TAGS

    A minimal plugin (seriously, it's laughably small) to take all tags from an Obsidian entry and safely move it into YAML frontmatter. ðŸ’¾ download.

    Installation and use

    Enter any Obsidian vault. Make sure "show hidden directories" is enabled or go there via terminal. Unzip the .zip file into the .obsidian/plugins/ directory:

    Enable "community plugins" in Obsidian:

    And enable the plugin.

    Now just type Command/Control-P to open the palette and choose "Move Tags (type "tags" and it should be right there)".
